- __Números__ - _int e double_
- __String__ - _String_ 
- __Booleano__ - _bool_ 

Todos os tipos básicos permitem o uso da notação `.` para acesso de funcionalidades, ex.:
```dart
main() {
	int n1 = (-5.67).abs();
	int n2 = -7.88;
	
	print(n1); //5.67
	print(n2.abs()); //7.88	
}
```
>É possível utilizar a notação, tanto por meio do valor direto, quanto por meio da variável.
